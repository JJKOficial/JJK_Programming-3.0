Web Site

Podemos considerar um Web site uma coleção de páginas HTML estáticas, ou seja, que não interagem com um banco de dados através de uma linguagem de servidor Web. Ou seja, aqui todo o conteúdo do site está escrito diretamente no documento HTML, assim como as imagens e outras mídias. Claro que, para qualquer página Web ser fornecida publicamente a mesma deve estar hospedada em um simples servidor Web (hospedagem de sites).

Aplicação Web

Uma aplicação Web pode conter uma coleção de páginas, porém o conteúdo destas páginas é montado dinamicamente, ou seja, é carregado através de solicitações (requisições) à um banco de dados, que conterá armazenado os textos e indicação dos caminhos das imagens ou mídias que a página precisa exibir.

Porém um HTML não tem acesso direto à um banco de dados, e esta comunicação deve ser feita por uma linguagem de programação de servidor Web.

Esta aplicação escrita com uma linguagem de servidor que tem o poder de acessar o banco de dados e montar a página HTML conforme o solicitado pelo navegador. Estas solicitações podem ser feitas de várias maneiras, inclusive utilizando JavaScript.

Portanto uma aplicação Web é mais complexa porque precisa de uma linguagem de servidor para poder intermediar as solicitações do navegador, um banco de dados, e muitas vezes (porém não Aplicação Web obrigatoriamente) exibir páginas HTML com estes conteúdos.

Exemplo de linguagens de servidor Web:

  • Java;
  • PHP;
  • Python;
  • Ruby ou Rails;
  • NodeJS etc...

Editores e IDES

1-Editores

7bz1 Os editores de texto são programas de computador leves e ideais para escrever e editar as páginas de um site, como Visual Studio Code (https://code.visualstudio.com/), Sublime (https://www.sublimetext.com/), Atom (https://atom.io/) e Notepad++ (https://notepad-plus-plus.org), para os dispositivos Android, pode- se utilizar o Acode (https://www.acode.com) ou (para baixar a app basta acessar a secção "apps e ficheiros de apoio") e entre outos que possuem realce de sintaxe e outras ferramentas para facilitar o desenvolvimento de páginas.

2-IDES

Há também IDEs (Integrated Development Environment) que são editores mais robustos e trazem mais facilidades para o desenvolvimento de aplicações Web, se integrando com outras funcionalidades. São alguns deles: WebStorm (https://www.jetbrains.com/webstorm/) Eclipse (https://www.eclipse.org/) e Visual Studio (https://visualstudio.microsoft.com).

Para não haver textos grandes ou matérias que muitos não gostam ler, recomendamos os seguntes cursos (em video aulas) no YouTube:


1.HTML e CSS

O HTML (Hypertext Markup Language) ou linguagem de marcação de hipertexto que foi desenvolvido para suprir a necessidade exibição de documentos científicos fornecidos por uma rede de Internet. Para termos uma comparação, é como se a Web fosse desenvolvida para exibir monografias redigidas e formatadas pela Metodologia do Trabalho Científico da ABNT. Porém, com o tempo e a evolução da Web e de seu potencial comercial, tornouse necessária a exibição de informações com grande riqueza de elementos gráficos e de interação.

Para acessar as video aulas, basta clicar o link HTML e CSS.

2.JavaScript (JS)

O Javascript é uma linguagem de script que incorporado nos tag's Html, permite incrementar a apresentação e interactividade das páginas Web.

É importante saber que o Javascript é completamente diferente do Java. Mesmo que os dois sejam utilizados para criar páginas Web evoluídas. Mesmo que os dois usam o mesmo termo Java (café em americano), temos aí duas ferramentas informáticas bem diferentes.

JavaScript

  • Código integrado na página Html;
  • Código interpretado pelo browser no momento da execução;
  • Códigos de programação simples mas para aplicações limitadas;
  • Permite de aceder aos objectos do browser;
  • Confidencialidade do código é nulo (Código é visível).

Java

  • Moduo (applet) distinto da página Html;
  • O código é compilado antes da sua execução;
  • Linguagem de programação muito mais complexa mas mais poderosa;
  • Não tem acesso aos objectos do browser;
  • Segurança (Código compilado).

Para acessar as video aulas basta clicares no link JavaScript.
